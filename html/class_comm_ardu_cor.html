CommArduCor Class Reference

The CommArduCor class wraps CommHTTP, CommUDP, and CommSerial into one object designed for communicating with ArduCor arduino samples. This object handles the discovery of both new and previously discovered arduinos, and handles sending packets and receiving packets from arduinos. More...
