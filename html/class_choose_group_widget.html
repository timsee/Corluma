ChooseGroupWidget Class Reference

The ChooseGroupWidget class is a full page widget that aids in choosing a specific group or room to edit. It uses EGroupAction to determine the state it intends to select a group to. There are slight UI differences depending on whether the group is being selected for editing or deleting. More...
