DiscoveryWidget Class Referenceabstract

The DiscoveryWidget class is a widget that gets displayed on the Discovery Page. Each subclass of this widget maps to a different communication stream. So theres a widget for Serial arduinos, a widget for Yun arduinos, and a widget for Hues. More...
