cor::StatusIcon Class Reference

The StatusIcon class is a very simple QWidget that displays an indicator of the current status of a device. If it is showing red, the device cannot be reached. If it showing black or white, it is off or on respectively. It also shows the current brightness as a shade of grey. More...
