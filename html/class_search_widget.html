SearchWidget Class Reference

The SearchWidget class is a widget with three main sections. The top contains an editable field with a plus and minus button. The middle is a list of objects that are currently being searched for. Objects are added to the searching list by clicking the plus button. The bottom is a list of objects that have been discovered or connected. An object cannot be in both the searching list and the connected list at the same time. More...
