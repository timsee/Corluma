ParentGroupWidget Class Reference

The ParentGroupWidget class is a widget that displays all the subgroups and lights associated with a parent. subgroups only go one level deep, so if a parent is "First Floor Lights" and this parent contains "Living Room" as a subgroup, and "Living Room" contains "Computer Desk" of a subgroup of it, then "Computer Desk" is treated as a subgroup of "First Floor Lights", not "Living Room".
