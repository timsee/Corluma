ChooseMetadataWidget Class Reference

The EditNameAndDescriptionWidget class allows the user to set the name and the description of a group, room, or mood. It provides two fields: a name field, which is required, and a description field, which is optional. The widget checks that the name and description are both shorter than a maximum length, and that the name isn't used by any reserved strings for the name such as "Miscellaneous". More...
