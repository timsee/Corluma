DummyShareUtils Class Reference

The DummyShareUtils class dervies from the PlatformShareUtils class and exists only to make it so that you don't need to wrap ShareUtils in preprocessor macros when compiling for desktop. When working with viewing and sending files on desktop, you should instead use QFileDialog.

#include "shareutils.hpp"
