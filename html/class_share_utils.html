ShareUtils Class Reference

The ShareUtils class interfaces the various ShareUtils for each platform with a standard Qt interface. This allows you to include a ShareUtils object in your application as a standard Qt object. The ShareUtils class abstracts away the native code and emits signals instead. For android and iOS this provides the ability to view and send files. For desktop builds, this only provides a dummy. In desktop environments this design pattern doesn't make sense and QFileDialog is preferred.

#include "shareutils.hpp"
