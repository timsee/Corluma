ListDevicesGroupWidget Class Reference

The ListDevicesGroupWidget class is a subclassed ListCollectionWidget that displays all the devices in a collection. It contains a widget at the top that gives info on the collection and the ability to edit it. It also contains an array of ListDeviceWidgets, which can be shown or hidden by clicking the top widget.
