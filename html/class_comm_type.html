CommType Class Reference

inherited by comm types, provides a general interface that can be used to do connections and sending packets. Each CommType also has its own conenctionList(), which lists up to 5 of the previous connections. This list persists in the application's memory after the application closes.

#include "commtype.h"

Inherits QWidget.

Inherited by CommHTTP, CommHue, CommSerial, and CommUDP.
