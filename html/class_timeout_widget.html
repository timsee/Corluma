TimeoutWidget Class Reference

The TimeoutWidget class provides a widget that controls whether or Timeout Mode is on or off. If it is on, X minutes after the last packet is sent the lights turn off. If its off, lights stay on indefinitely. This widget also provides an interface to choose how many minutes a light should stay on.
