MultiColorPicker Class Reference

The MultiColorPicker class is a color picker designed for choosing multiple colors at once. In contrast to the SingleColorPicker, which polls the ColorWheel for colors, this polls the position of the your mouse/touch event and sets multiple colors programmatically. More...
