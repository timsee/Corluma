ListMoodWidget Class Reference

The ListMoodGroupWidget class is a subclass of a ListCollectionWidget. It displays a collection of moods. It contains a widget at the top that displays all the information about the collection and gives the ability to edit it. The rest of the collection is a series of buttons where each one represents a mood that contains the same devices as the collection. More...
