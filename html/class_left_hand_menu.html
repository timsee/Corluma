LeftHandMenu Class Reference

The LeftHandMenu class is a left hand drawer that can either be pulled in or out when the app has a portrait aspect ratio. If it is a landscape aspect ratio, the drawer is permanently out. This menu contains buttons for all the main pages of the app, as well as the ability to choose specific lights to control. More...
