UPnPDiscovery Class Reference

The UPnPDiscovery class listens for UPnP packets and signals them out when they are received. Rather than using the standard startup/shutdown methods that most of the backend classes use in this app, this object automatically turns on and off its socket based on how many listeners are requesting that it sends UPnP packets.
