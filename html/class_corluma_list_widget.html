CorlumaListWidget Class Reference

The CorlumaListWidget class is an instance of a QScrollArea designed to hold multiple ListCollectionWidgets in a scroll area that extends vertically. This widget also handles cases where the ListCollectionWidgets grow and shrink dynamically. More...
