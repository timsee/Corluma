HSVSliders Class Reference

The HSV class is a set of 3 sliders, one representing hue values, one representing saturation values, and one representing value values.These sliders are used by the ColorPicker during its HSV layout to give a second option and represnetation of the HSV color being chosen by the ColorPicker.
