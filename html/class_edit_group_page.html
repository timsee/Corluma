EditGroupPage Class Reference

The EditGroupPage class is a page that is used to edit existing groups of lights. It can edit both collections (paths to lights without any colors or states saved) and moods (both a path to a light and its color and states are saved). It allows you to remove and add devices from a group, as well as change its name. You can only add and remove devices from this page, you cannot change the settings of individual devices. A special case of this page can also create new collections and moods. More...
