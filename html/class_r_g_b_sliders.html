RGBSliders Class Reference

The RGBSliders class is a set of 3 sliders, one representing red values, one representing green values, and one representing blue values.These sliders are used by the ColorPicker during its standard layout to give a second option and represnetation of the RGB color being chosen by the ColorPicker.
