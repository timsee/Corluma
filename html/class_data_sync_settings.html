virtual void cleanupSync ()=0
 cleanupSync After the sync is complete, certain actions need to be ran. For example, Hues require a schedule to be kept synced to timeout properly. The cleanup thread starts after the ArduinoDataSync to run the routines needed to keep data in sync in the long term. This function contains all those routines.
