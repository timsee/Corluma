DropdownTopWidget Class Reference

The DropdownTopWidget class provides a widget that can be used at the top of a dropdown list. It is frequently used in the LightPage and the MoodPage to hide the details of overally lights/moods but still display some meta information. It also provides the user with buttons to edit the contents of the widget, or to select all/none of the contents in some cases. More...
