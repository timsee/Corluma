SingleLightBrightnessWidget Class Reference

The SingleLightBrightnessWidget controls the brightness of a single light. This replaces the global brightness widget on widgets where only one light is selected at time. It shows the light's current color on the left, and then provides a slider for changing hwo bright it is. More...
