nano::LeafLightInfoWidget Class Reference

The LeafLightInfoWidget class is a widget made to be displayed in lists that contains the name of a light and much of its hardware information such as its model number and software version. The user can also edit the name of light from this widget. If an edit is made, this widget signals out the new name for it to be sent to the bridge. More...
