GlobalBrightnessWidget Class Reference

The GlobalBrightnessWidget is a widget that controls the brightness of all selected lights. It is set to display the brightness of all lights averaged together, and when it changes, it sets all lights to tis new brightness. It also has an on/off switch. More...
