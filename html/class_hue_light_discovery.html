HueLightDiscovery Class Reference

The HueLightDiscovery class is a widget that provides a UI for discovering new hue lights. The Bridge gets its scanning state turned on for automatic discovery. The user can also input serial numbers for the bridge to use for manual discovery. All new lights found from this discovery state are added to the widget in a list. More...
