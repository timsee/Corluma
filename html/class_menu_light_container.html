MenuLightContainer Class Reference

The MenuLightContainer class is a container widget that displays mulitple ListLightWidgets. This widget controls its own height so that it can properly display enough lights. The widget signals the unique ID of a light when a light is clicked. Lights can be highlighted when selected. More...
