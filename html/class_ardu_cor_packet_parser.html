ArduCorPacketParser Class Reference

The ArduCorPacketParser class takes the packets used for communication with other devices as input, parses them, and emits signals based on the contents of a packet. For example, a main color change packet will emit a receivedMainColorChange(int, QColor) signal. This class is used to emulate the functionality of the parser that is embedded on the arduino platforms on devices like the Philips Hue. More...
