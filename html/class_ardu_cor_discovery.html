ArduCorDiscovery Class Reference

The ArduCorDiscovery class runs the discovery routines for ArduCor controllers. It does not actually connect to serial ports or sockets. Instead it uses existing CommHTTP, CommUDP, and CommSerial to handle the communication protocols. Instead, it stores, saves, and verifies discovery packets. More...
