DataSyncTimeout Class Reference

The DataSyncTimeout class syncs timeout settings for lights that need it. Not all lights support timeouts out of the box, and some need to mock the functionality with a schedule that runs an action that turns the light off. This DataSync runs slower than other data syncs, and does not need to be in sync for widgets to show that everything is in sync. More...
