The GroupData class manipulates and reads a JSON representation of multiple lights. This data can be exported or overriden from the SettingsPage. The data stored here is only the data that can&#39;t be stored externally. For example, Hue Bridges support groups and rooms, so all hue related data is stored on the bridge.
