DiscoveryHueWidget Class Reference

The DiscoveryHueWidget class helps in discovering a Hue Bridge. It attempts to find a bridge automatically, and if attempts fail, it prompts the user to enter the IP address. It also prompts the user through setup steps such as pressing button on the bridge. More...
