What is "Advance Mode"?

Advance Mode turns on the ability to use open-source arduino LED projects. Currently the only supported project is: ArduCor. Corluma provides full support for provides full all samples in ArduCor, although the serial samples don't work on mobile builds of Corluma since... well... theres very limited serial support for phones.
